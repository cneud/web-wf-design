Taverna Lite Workflow Designer

Welcome to the Taverna Lite Workflow Designer!

The application is based on the WireIt library. For the latest WireIt code and their official documentation see the WireIt Web site.

This is a prototype designed to show how existing workflows can be connected together and run using the Taverna Command Line Tool and later the Taverna Server.

This is NOT a replacement for the Taverna Workbench and its Workflow Editor but as a support to to go on top of pre existing workflows.

It is intended to hide much of the complexity required to build Taverna workflows including:

  • Hides XML Splitters and Merges
  • Hides Cross and Dot product choices
  • Hides complex Beanshells
  • Hides looping
  • Avoid users having to decide which parameters are needed for a Web service
What it does do includes:
  • Allow several workflows to be joined together
    • Even when the output name in one workflow is not the same as the input name as another workflow
  • Provides a simple GUI for running a workflow

For administrators and developers

Documentation for administrators and developers interested in installing the application, looking at the source code or extending the existing code.

Current state/Limitations

  • Setting individual inputs is limited to workflows where all inputs are depth 0 or 1
    • Input can be a string
    • Input can be from a URL
    • Input can be a list of strings
    • File input is not yet supported
  • Baclava inputs are supported
    • From a URL
    • File Baclava not yet supported
  • Baclava output is supported
    • As a URL
    • Fancy Baclava Viewer for displaying Baclava files in a Web browser
    • Baclava output from one workflow can be used as Baclava input for another workflow
      • If and only if the input port names are a subset (or equal to) the ouput port names
  • Individual output support by casting to String
  • List output supported by flattening lists of lists and casting to string
  • Only predefined workflows
    • This is mainly for security
    • Additional workflows can easily be added
