Welcome to Taverna Lite Workflow Designer

This is a demo of the Taverna Lite Workflow Designer built on the WireIt library.


What this example does:

  • * Defines a visual language for "wirings"/workflows with inputs, outputs and a number of Taverna components.
  • * Allows Taverna components to be connected to inputs and outputs.
  • * Allows Taverna components to be connected even if port names do not match.
  • * Provides access to Baclava XML output of a connected workflow.
  • * Allows the "wiring"/workflow to be sent to the server to run.

How to use it:

  • * Drag and drop components from the Module list to the working canvas.
  • * Create a wiring with inputs, outputs and Taverna components (or load an existing one).
  • * Press the "Run" button to run the code on the Taverna Server.

Warning: Composed "wirings"/workflows are shared anonymously.
Someone could change/delete your wirings or create evil ones!
