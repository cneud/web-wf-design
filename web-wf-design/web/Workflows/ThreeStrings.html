Description

This is a simple test workflow which just passes the three inputs to the three outputs with the same name.

Note the extra "in_" and "out", have been added because WireIt can not handle duplicate names between inputs and outputs..

Inputs

Left, Middle, Right

Type

This can be any object.

Outputs

Left

Type

This will be the same type as (input) Left.

Expected Value

Exactly the same as (input) Left

Middle

Type

This will be the same type as (input) Middle.

Expected Value

Exactly the same as (input) Middle

Right

Type

This will be the same type as (input) Right.

Expected Value

Exactly the same as (input) Right
