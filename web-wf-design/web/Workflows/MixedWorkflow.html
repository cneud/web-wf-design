Description

This is a test workflow to show lists working.

  • Each value of LeftList is concatenated to the single value of LeftNoList.
  • Eacb value of RightList is ooncatenated to the single value of RightNoList.
  • Then each value of the left results is concatenated to each value of the right results.

Note the .(list) annotation is not part of the port name but only a reminder that is is of depth 1 (list).

Inputs

LeftList, RightList

Type

List of Strings. (A single value will be automatically wrapped into a list of length 1)

LeftNoList, RightNOList

Type

A String.

Outputs

Result

Type

A list of Strings. Or actually a list of lists of Strings.

Expected Number of Values

The product of the number for each input

Expected Value

A Contacentation including one value from each list and each Singelton.
